## 🐱 API

This repo contains the API for the Codecademy BE challenge. The API is built using TypeScript, Node.js and Express.js. The API is deployed and can be accessed at https://floral-wave-4366.fly.dev/api-docs/.

## 🚀 Getting Started
```bash
# Clone the repo
git clone
yarn
yarn compile
yarn start
```

<!-- development -->
